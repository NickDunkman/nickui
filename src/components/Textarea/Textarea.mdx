# `<Textarea>`

Textarea is a form control that allows users to enter & edit text.

## Textarea-specific props

Textarea takes some custom props, plus all the props that can be used with
[&lt;textarea&gt;](https://react.dev/reference/react-dom/components/textarea).

These are handled via the `value`, `defaultValue`, & `disabled` props.

## Field layout

Textareas can optionally have each of the components of a standard
[Field](/docs/forms-field--docs): label, explainer, hint, and error. Textarea
automatically handles the accessibility of these pieces for you!

<Canvas of={Stories.Field} sourceState="shown" />

## Sizing

Sizing is done via the `sizer` prop. You can pass "small", "medium", or "large".
In keeping with the mobile-first philosophy of Tailwind, the default size is
"small".

<Canvas of={Stories.AllSizes} sourceState="shown" />

## Controlling the height

Minimum vertical sizing of textareas is done via the `rows` prop (the default is
2 rows).

<Canvas of={Stories.MinimumHeight} sourceState="shown" />

To allow the Textarea to automatically expand up to some maximum number of rows
(default is `Infinity`), use the `autoResize` and `maxRows` props. Try adding or
removing text from these examples!

<Canvas of={Stories.AllResizables} sourceState="shown" />

To prevent the user from being able to resize the Textarea manually, use the
`disableManualResize` prop.
