# `<Textarea>`

A form control that allows users to enter & edit a mass of text.

## Control states

These are handled via the `value`, `defaultValue`, & `disabled` props.

<Canvas of={Stories.AllControlStates} sourceState="shown" />

## Controlling the height

Minimum vertical sizing of textareas is done via the `rows` prop (the default is
2 rows). The Textarea will automatically grow (& shrink) up to the `maxRows`
prop to accomodate the value (the default is 10 rows).

<Canvas of={Stories.AutoResizableHeight} sourceState="shown" />

For a fixed height, use the same value for `rows` and `maxRows`.

<Canvas of={Stories.FixedHeight} sourceState="shown" />

To prevent the user from being able to resize the Textarea manually, use the
`disableManualResize` prop.

<Canvas of={Stories.DisableManualResize} sourceState="shown" />

## Field layout

Textareas can optionally have each of the components of a standard
[Field](/docs/forms-field--docs): label, explainer, hint, and error. Textarea
automatically handles the accessibility of these pieces for you!

<Canvas of={Stories.FieldLayout} />

## Sizing

Sizing is done via the `sizer` prop. You can pass "small", "medium", or "large".
In keeping with the mobile-first philosophy of Tailwind, the default size is
"small".
