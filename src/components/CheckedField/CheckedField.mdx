Wraps a single checkable form control with a standard layout (label & hint, to
the right of the control).

This is simpler than the <Docs of="Field" /> component. There’s no `error`
message, since if one or both possible values is invalid, the field has no
purpose. And there’s no `required` state, since that would make the unchecked
state invalid.

<div>
  Note: this layout is already built-into <Docs of="Checkbox" />,{' '}
  <Docs of="Radio" />, and <Docs of="Switch" />. You can use this component to
  create a custom form control that behaves the same way.
</div>
