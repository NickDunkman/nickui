## Field layout

TextInputs can optionally have each of the components of a standard
[FieldLayout](/?path=/docs/components-fieldlayout--docs): label, description,
hint, and error. TextInput automatically handles the accessibility of these
pieces for you!

<Canvas of={Stories.FieldLayout} sourceState="shown" />

## Sizing

Sizing is done via the `sizer` prop. You can pass "small", "medium", or "large".
In keeping with the mobile-first philosophy of Tailwind, the default size is
"small".
