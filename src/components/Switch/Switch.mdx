# `<Switch>`

A variation of the [Checkbox](/docs/forms-checkbox--docs) component that has a
more explict-feeling "off" state. Whereas a group of co-located Checkbox fields
feels more like multiple options for one field, a group of Switch fields feels
more like discrete fields under a common category.
