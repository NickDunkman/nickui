## Controlling the height

Minimum vertical sizing of textareas is done via the `minRows` prop (the default
is 2 rows). The <Docs of="TextArea" /> will automatically grow up to the
`maxRows` prop to accomodate the value (the default is 10 rows). For an
unbounded height, you can use `maxRows={Infinity}`.

<Canvas of={stories.autoResizableHeight} sourceState="shown" />

For a fixed height, use the `rows` prop.

<Canvas of={stories.fixedHeight} sourceState="shown" />

To prevent the user from being able to resize the <Docs of="TextArea" />
manually, use the `disableManualResize` prop.

<Canvas of={stories.disableManualResize} sourceState="shown" />
