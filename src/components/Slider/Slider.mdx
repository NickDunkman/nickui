# `<Slider>`

A form control that allows users to enter & edit a single line of text.

## Control states

These are handled via the `value`, `defaultValue`, & `disabled` props.

<Canvas of={Stories.AllControlStates} sourceState="shown" />

## Customizing the value range

Use the `min`, `max`, and `step` props to customize the bounds of the range, and
the granularity of values within it. You can also use `shiftSteps` to control
how many steps are moved when the user is holding the shift key while using the
keyboard to interact with the Slider.

<Canvas of={Stories.CustomRange} sourceState="shown" />

## Field layout

Text can optionally have each of the components of a standard
[Field](/docs/forms-field--docs): label, explainer, hint, and error. Text
automatically handles the accessibility of these pieces for you!

<Canvas of={Stories.FieldLayout} sourceState="shown" />

## Sizing

Sizing is done via the `sizer` prop. You can pass "small", "medium", or "large".
In keeping with the mobile-first philosophy of Tailwind, the default size is
"small".
