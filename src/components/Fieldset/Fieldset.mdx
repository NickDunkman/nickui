# `<Fieldset>`

Fieldset wraps multiple form controls in a standard field layout (label, hint,
error message, etc).

<FakeCanvas>
  <Fieldset
    label="A label"
    explainer="An explainer"
    error="An error message!"
    hint="A hint"
  >
    <div className="flex flex-col gap-3">
      <FormControlPlaceholder>One form control</FormControlPlaceholder>
      <FormControlPlaceholder>Another form control</FormControlPlaceholder>
      <FormControlPlaceholder>And so on ...</FormControlPlaceholder>
    </div>
  </Fieldset>
</FakeCanvas>

## Props

In addition to the commonly used props below, you can pass any prop accepted by
the [&lt;fieldset&gt;](htps://react.dev/reference/react-dom/components/common)
element, which will be applied to the Field's root element.

<ArgTypes of={Fieldset} sort="requiredFirst" />

## Basic elements

These elements are controlled using the `label`, `explainer`, and `hint` props.

<Canvas of={Stories.Basic} sourceState="shown" />

## Sizing

Sizing is done via the `sizer` prop. You can pass "small", "medium", or "large".
In keeping with the mobile-first philosophy of Tailwind, the default size is
"small".

<Canvas of={Stories.AllSizes} />

## Errors

To add an error message, use the `error` prop.

## Required fields

The `required` prop is used to add an asterisk to the label.
